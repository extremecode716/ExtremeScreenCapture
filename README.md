# ExtremeScreenCapture


### 사용법 요약
실행파일 폴더(다운로드) -> ExtremeScreenCapture_env.txt (설정) -> ConsoleScreenCapture.exe (실행) -> 단축키 입력 (Ctrl + 1 또는 Ctrl + 2)

***

### 커스터마이징
자유롭게 커스터마이징 허용

#### 주의사항
Key Hook 방식으로 단축키를 받기 때문에
exe 파일이 활성화 상태가 아닐 때도 단축키를 입력받을 수 있다.
( 백신 검사에서 제외해야 실행 가능 )
